# Job Application Tracker

A modern web application that helps users organize and track their job applications in one place. The application allows users to manage application details, monitor progress through different hiring stages, and keep track of important follow-up dates.

## Purpose

Managing multiple job applications manually can become difficult and confusing. This project was built to provide a simple and organized way to track application progress, upcoming follow-ups, and hiring outcomes in a single place.

## Features

* Add new job applications
* Edit existing applications
* Delete applications
* Search applications by company name or job role
* Filter applications based on status
* Track application dates
* Track follow-up dates
* View upcoming follow-ups
* Dashboard statistics for application status
* Color-coded status indicators
* Data persistence using browser Local Storage
* Responsive and user-friendly interface

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/mundrumeghana04/Meghana-JobApplicationTracker.git
```

### Run Locally

1. Open the project folder.
2. Navigate to the `frontend` directory.
3. Open `index.html` in any modern web browser.

No additional installation or dependencies are required.
